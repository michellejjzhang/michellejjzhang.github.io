For Glory

Based on the popular video game, Super Smash Brothers, my group made a project that would allow a Slack workspace to have a leaderboard for the game. I worked on the Slack bot to add onto workspaces that would correspond to a website. The bot had various controls such as privately challenging individuals or having an open invite for casual games. A player could report back their score which would update the website's leaderboard.

  • Technologies Used: Node.js; Stdlib's API for Slack
  • Date: November 2017
  • Awards Won:
    Best Use of Stdlib API
    Best Mobile Multi-Person Game Platform
  • GitHub Link: https://github.com/amccannv/smashbot
  • DevPost Link: https://devpost.com/software/for-glory
  • Type of Project: Hackathon
